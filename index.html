EVL- Auto

หัวข้อประเมิน

1คุณภาพสินค้า / บริการ (Quality)
1.1สินค้าหรือบริการที่ได้รับตรงตามข้อกำหนดและมาตรฐาน
1 = สินค้า/บริการมีข้อบกพร่องชัดเจน · 2 = ตรงตามข้อกำหนดทั่วไป · 3 = ดีกว่าข้อกำหนด มีมาตรฐานสูง
1.2ความสม่ำเสมอของคุณภาพในการส่งมอบแต่ละครั้ง
1 = คุณภาพแตกต่างมากแต่ละครั้ง · 2 = คุณภาพใกล้เคียงกัน · 3 = คุณภาพคงเส้นคงวา ไม่มีข้อบกพร่อง
2การส่งมอบ (Delivery)
2.1ความตรงต่อเวลาในการส่งมอบสินค้า/บริการ
1 = ส่งล่าช้า >3 ครั้ง/ไตรมาส · 2 = ล่าช้า 1-2 ครั้ง · 3 = ส่งตรงเวลาหรือก่อนกำหนด
2.2ปริมาณสินค้า/บริการที่ส่งมอบตรงตามที่สั่ง
1 = ปริมาณผิดพลาด >20% · 2 = ผิดพลาดน้อยกว่า 10% · 3 = ปริมาณถูกต้อง 100%
3ราคา / ความคุ้มค่า (Cost)
3.1ราคาสินค้า/บริการมีความสมเหตุสมผลและแข่งขันได้
1 = ราคาสูงกว่าตลาด >15% · 2 = ราคาใกล้เคียงตลาด · 3 = ราคาต่ำกว่าตลาด มีโปรโมชัน
4การบริการและการสื่อสาร (Service)
4.1การตอบสนองและสื่อสารเมื่อมีปัญหาหรือข้อร้องเรียน
1 = ใช้เวลา >3 วันทำการ · 2 = ตอบภายใน 1-2 วัน · 3 = ตอบภายในวันเดียวกัน
4.2ความสามารถในการแก้ไขปัญหาหรือข้อบกพร่อง
1 = ปัญหาไม่ได้รับการแก้ไข · 2 = แก้ไขบางส่วนหรือล่าช้า · 3 = แก้ไขได้ครบถ้วน รวดเร็ว

ข้อมูลผู้ประเมิน

Supplier กรอกรหัสพนักงานและเลือกงานที่ต้องการประเมิน Buyer

หัวข้อประเมิน Buyer

1กระบวนการจัดซื้อและการสื่อสาร
1.1ความชัดเจนของข้อกำหนดและเงื่อนไขการจัดซื้อ
1 = ข้อกำหนดขาดรายละเอียด ต้องสอบถามเพิ่มมาก · 2 = ข้อกำหนดพอเข้าใจ · 3 = ครบถ้วน ชัดเจน ไม่ต้องสอบถามเพิ่ม
1.2ความรวดเร็วในการตอบคำถามและให้ข้อมูล
1 = ตอบช้ากว่า 3 วัน · 2 = ตอบภายใน 1-2 วัน · 3 = ตอบภายในวันเดียวกัน
2การชำระเงินและเอกสาร
2.1ความตรงต่อเวลาในการชำระเงินตามสัญญา
1 = ชำระล่าช้า >15 วัน · 2 = ล่าช้า 1-7 วัน · 3 = ชำระตรงตามกำหนด
2.2ความถูกต้องของเอกสาร PO และสัญญา
1 = เอกสารผิดพลาดหลายจุดต้องแก้ซ้ำ · 2 = มีข้อผิดพลาดเล็กน้อย · 3 = เอกสารถูกต้อง ครบถ้วน
3ความเป็นธรรมและความโปร่งใส
3.1กระบวนการคัดเลือกและตัดสินใจมีความโปร่งใส
1 = ไม่มีการแจ้งเหตุผลชัดเจน · 2 = มีการแจ้งผลแต่ไม่ละเอียด · 3 = มีการสื่อสารครบถ้วน ชัดเจน

GCP- Auto

หัวข้อประเมิน เกณฑ์ GCP

1ระบบบริหารคุณภาพ (Quality Management System)
1.1มีระบบควบคุมคุณภาพและเอกสารประกอบอย่างครบถ้วน
1 = ไม่มีเอกสาร/ระบบ QC · 2 = มีบางส่วนแต่ไม่สมบูรณ์ · 3 = มีระบบ ISO หรือเทียบเท่า
1.2การทดสอบและการรับรองผลิตภัณฑ์ตามมาตรฐานสากล
1 = ไม่มี Certification · 2 = มี Cert บางส่วน · 3 = ผ่าน ISO/TUV/CE ครบถ้วน
2ความสามารถในการผลิตและกำลังการผลิต (Capacity)
2.1กำลังการผลิตและความยืดหยุ่นในการรองรับปริมาณสั่ง
1 = Capacity <70% ของที่ต้องการ · 2 = Capacity 70-100% · 3 = มี Capacity สำรอง >20%
2.2เทคโนโลยีและเครื่องจักรที่ใช้มีความทันสมัย
1 = เครื่องจักร >15 ปี ไม่ได้อัพเกรด · 2 = อัพเกรดบ้าง · 3 = เทคโนโลยีล่าสุด อัตโนมัติ
3สถานะทางการเงินและความมั่นคง (Financial)
3.1ความมั่นคงทางการเงินและสภาพคล่องขององค์กร
1 = มีปัญหาสภาพคล่อง/หนี้สูง · 2 = สถานะปานกลาง · 3 = สถานะทางการเงินดีเยี่ยม
4ความรับผิดชอบต่อสังคมและสิ่งแวดล้อม (CSR/ESG)
4.1นโยบายและการปฏิบัติด้านสิ่งแวดล้อม
1 = ไม่มีนโยบาย ESG · 2 = มีนโยบายแต่ไม่มีการปฏิบัติจริง · 3 = มีนโยบายและหลักฐานการปฏิบัติ
4.2การปฏิบัติตามกฎหมายแรงงานและสิทธิมนุษยชน
1 = มีประวัติละเมิดกฎหมายแรงงาน · 2 = ปฏิบัติขั้นพื้นฐาน · 3 = ปฏิบัติครบ มีการตรวจสอบ
